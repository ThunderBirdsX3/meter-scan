# UI Deepen v2.1 — Design + Implementation Plan

User feedback: UI plain, all Ionic defaults. Goal: modern, beautiful, both light/dark.
Decisions (user-approved 2026-07-05): keep teal/emerald palette (deepen, not replace) · duotone filled vehicle icons · all pages in scope.

## 1. Token additions (DS-Tokens v2.1 — additive only)

New primitives/semantics in `src/theme/variables.scss`:

| Token | Light | Dark | Use |
|---|---|---|---|
| `--gradient-brand` | 135deg teal-700 → emerald-700 | 135deg teal-600 → emerald-600 | hero cards, entry-detail header. White text only (large/bold ≥ AA) |
| `--color-surface-app` | `#f3f7f6` | `#0c100f` | page canvas — cards float on tinted bg |
| `--color-surface-brand-subtle` | `rgba(15,118,110,.08)` | `rgba(20,184,166,.14)` | icon avatars, active tab pill, segment active |
| `--radius-2xl` | 16px | — | cards, grouped lists |
| `--radius-3xl` | 20px | — | hero card |
| `--shadow-card` | layered soft (2 layers, ≤ .07 alpha) | `none` (border `--color-border-default` instead) | raised cards |

Rewire: `--ion-background-color` → `--color-surface-app` (both modes). Items/cards stay `surface-raised`.

## 2. Global restyle (`global.scss`)

- Toolbars transparent (`--ion-toolbar-background: transparent` on pages), bold large titles.
- Grouped inset lists: `ion-list[inset]`-style treatment — radius-2xl, shadow-card, margin space-4.
- Tab bar: no top border, elevated, active tab gets tinted pill (`surface-brand-subtle`) behind icon+label.
- Buttons: `--border-radius: 12px`. Segments: pill track + white/raised active thumb.

## 3. Per-page

- **stats** — gradient hero card w/ headline stat (white bold text), below: 2-col stat tile grid (raised cards, tabular-nums).
- **add** — form grouped into raised cards, block CTA prominent, section labels `--tracking-label` uppercase.
- **history** — entry rows as cards; vehicle icon in tinted circle avatar (40px, `surface-brand-subtle` bg, primary tint icon); amount right-aligned tabular-nums semibold.
- **entry-detail** — gradient hero header (vehicle + date + big liters/amount readout), detail rows in grouped card.
- **settings / trips / vehicles** — iOS grouped inset lists; leading icons in 32px rounded-square tinted containers; vehicles picker uses duotone icons.
- **tabs** — per §2.

## 4. Vehicle icons — `src/assets/vehicle-icons/*.svg`

8 icons redrawn: motorcycle, bigbike, scooter, sedan, suv, ppv, van, truck.
- viewBox `0 0 24 24` kept — zero code changes at call sites.
- ~~v2 duotone (solid body + opacity .35 detail)~~ superseded same day — device feedback: read as cartoonish.
- **v3 outline (final)**: `stroke="currentColor"` width 1.5, round caps/joins, `fill="none"`; tiny filled accents for legibility (bike seats, bigbike tank). Matches ionicons-outline used in settings rows.
- `currentColor` → inherits text color = auto light/dark correct; avatar context tints to primary.
